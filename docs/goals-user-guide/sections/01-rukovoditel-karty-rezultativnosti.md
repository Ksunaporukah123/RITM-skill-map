# Руководитель: карты результативности

**Предусловия:** авторизация в РИТМ; роль «Руководитель».

**Переход:** **Целеполагание** → **«Карты результативности»** (или личный кабинет → **«Моя команда»**).

---

## Создание КР для сотрудника

1. Найдите сотрудника. В столбце **«Карты результативности, периоды»** нажмите **бейдж** периода («2026 год», «1Q2026»).
2. В поповере — **«Создать КР»**.
3. Заполните карту:
   - **Добавить КПЭ** → выбор из списка (ГЦ сотрудника, кроссфункциональные);
     - **«Доступно»** / **«Недоступно»** по периоду;
     - значок **«Глаз»** — детали;
     - чекбоксы → **«Выбрать КПЭ»** → **«Сохранить»**;
     - в карте: **вес**, **вид**, **зона ответственности**;
   - **Добавить ЗРД** — наименование, вес, ожидаемый результат;
   - **Добавить ККПЭ** — как КПЭ, вид «Корректирующий» автоматически.
4. **«Сохранить»** внизу формы.

**Результат:** статус **«Черновик»**.

**Важно:** сумма весов КПЭ и ЗРД = **100%**; удаление — **«✕»**; при закрытии без сохранения — запрос подтверждения.

---

## Переиспользование КР

1. Бейдж периода → **«Открыть КР»** → меню **«…»** → **«Переиспользовать карту результативности»**.
2. **«Выбрать сотрудников»** — только **«Доступно»** (нет целей на период).
3. **«Сохранить»** → **«Создать карты результативности»** → **«Создать»** → **«Закрыть»**.

**Результат:** копии в **«Черновик»** для выбранных сотрудников.

---

## Просмотр КР подчинённых

1. Фильтр **«Период»** — год.
2. **«Сотрудники прямого подчинения»** — прямые / всё подразделение.
3. Поиск по **ФИО**, фильтры: ГЦ, должность, подразделение.
4. Бейдж периода → поповер → **«Открыть КР»** (Черновик, Разработка, Активная, Архивная).

---

## Редактирование в статусе «Черновик»

1. **«Открыть КР»** → статус **«Черновик»**.
2. Измените период, срок, КПЭ, ЗРД, ККПЭ.
3. **«Сохранить»** — карта остаётся в черновике.

Для согласования отдельно: **«Отправить на акцептование»**. Меню **«…»**: карта руководителя, история, переиспользование.

---

## Отправка на акцептование (Черновик → Разработка)

1. Проверьте данные и **веса = 100%**.
2. **«Отправить на акцептование»**.
3. Статус **«Разработка»**; появится **«Редактировать КР»**.

---

## Редактирование в «Разработка» / «Активная»

1. **«Редактировать КР»** → подтвердите возврат в **«Черновик»**.
2. Внесите правки → **«Сохранить»** → при необходимости снова **«Отправить на акцептование»**.

**Важно:** согласование сбрасывается; история — **«…» → история карты**.

---

## Утверждение оценки результативности

**Предусловия:** КР **«Активная»**; в ПФК план утверждён, факт верифицирован.

1. Откройте активную КР.
2. **«Выполнение (оценка руководителя)»** и **«Оценка руководителя, %»** → **«Сохранить»**.
3. **«Утвердить оценку»** → процесс завершён.

# Делегирование полномочий

**Переход:** **Целеполагание** → вкладка **«Делегирование»**.

## Подвкладки

| Подвкладка | Содержание |
|------------|------------|
| **Мои роли** | Ваши роли и перечень действий; роли, делегированные вам |
| **Управление делегированием** | Назначение роли сотруднику, история |

## Назначение делегирования

1. **Управление делегированием** → выберите **роль** и **сотрудника**.
2. Укажите период **с** / **по** (пустое «по» — до текущего момента).
3. **«Сохранить»**.

При делегировании передаётся **весь функционал роли** (все действия роли).

## Завершение делегирования

В **истории делегирований** — **«Завершить»** для активной записи.

## Роли в системе (примеры)

- Линейный руководитель — создание и ведение КР, КПЭ, оценка;
- Руководитель группы целеполагания (РГЦ) — утверждение КР и оценок по ГЦ.

Подробный перечень действий отображается в блоке **«Мои роли»** при раскрытии роли.

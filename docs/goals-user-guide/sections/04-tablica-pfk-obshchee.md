# Таблица ПФК: общие элементы интерфейса

**ПФК** — таблица плановых, фактических и критических значений по КПЭ из карт результативности.

**Переход:** **Целеполагание** → **«Таблица ПФК»** (или **«ПФК»** / ожидаемое действие в личном кабинете).

## Элементы управления

| Элемент | Назначение |
|---------|------------|
| **Полный вид** | Формула, аналитики, руководитель ГЦ, верификатор |
| **Режим редактирования** | Ввод и утверждение; **«Сохранить»**, **«Закрыть»** |
| **Экспорт/Импорт** | Excel (краткий вид) |
| **Действие** (в строке) | История операций по КПЭ |
| Фильтры в заголовках | Статусы, период, зона ответственности |

## Столбцы таблицы

ID, наименование КПЭ, ед. изм., период, зона ответственности, **План**, **База/Крит. откл.**, **Статус плана**, **Факт**, **Статус факта**, % выполнения.

## Цепочка статусов

**План:** ожидание ввода → (Аналитик ПЛАН) → ожидание утверждения → (Руководитель ГЦ) → план утверждён.

**Факт:** ожидание ввода → (Аналитик ФАКТ) → ожидание утверждения → (Верификатор) → факт утверждён → расчёт выполнения.

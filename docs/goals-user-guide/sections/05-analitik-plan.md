# Аналитик ПЛАН

**Предусловия:** роль «Аналитик ПЛАН»; КПЭ со статусом плана **«Ожидание ввода плана»** (КР в «Разработка» или «Активная»).

## Ввод плановых значений

1. **«Необходимо внести плановые значения КПЭ»** или вкладка **ПФК**.
2. **«Полный вид»**, **«Режим редактирования»**.
3. Фильтр **«Ожидание ввода плана»**.
4. Заполните **План** и **База/Крит. откл.** → **«Сохранить»**.

**Результат:** **«Ожидание утверждения плана»** → Руководитель ГЦ.

## Редактирование плана

| Исходный статус | После сохранения |
|-----------------|------------------|
| Ожидание утверждения плана | Значение обновлено, статус без изменений |
| План утверждён | Диалог подтверждения → статус снова **«Ожидание утверждения плана»** |

## Импорт из Excel

1. **Экспорт/Импорт** → **«Экспорт Таблица ПФК краткий вид»** → `Таблица ПФК краткая форма.xlsx`.
2. Заполните только **«План»** и **«База/ Крит.откл»** (не меняйте ID и структуру).
3. **«Импорт Таблица ПФК краткий вид»** → *«Успешно. Таблица обновилась!»*.

**Советы:** применяйте фильтры перед экспортом; импорт перезаписывает значения в файле.

## Ключевые моменты

- Изменения вступают в силу после **«Сохранить»**.
- **«Закрыть»** — без сохранения.
- История — кнопка **«Действие»** в строке.

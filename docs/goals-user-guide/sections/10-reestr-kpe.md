# Реестр КПЭ

**Реестр КПЭ** — единый каталог ключевых показателей эффективности для карт результативности и таблицы ПФК.

**Переход:** **Целеполагание** → **«Реестр КПЭ»**.

## Возможности

- просмотр и поиск показателей;
- фильтры в заголовках столбцов;
- **«Полный вид»** — расширенный набор полей;
- **«Экспорт»** / экспорт выбранных;
- **«+ Создать»** — новый КПЭ (при наличии прав).

## Связь с ПФК

Реестр — **источник истины** для наименований и атрибутов КПЭ. Показатели из КР попадают в **таблицу ПФК** для ввода плана и факта. Изменения в реестре отражаются в связанных записях ПФК.

## Рекомендации

- Перед созданием КР проверяйте наличие нужного КПЭ в реестре.
- Согласуйте период оценки КПЭ с периодом карты результативности.

См. также: `04-tablica-pfk-obshchee.pdf`, руководство для **Руководителя** (добавление КПЭ в КР).

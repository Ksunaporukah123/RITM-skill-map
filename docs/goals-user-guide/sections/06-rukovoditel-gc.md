# Руководитель ГЦ: утверждение плана в ПФК

**Предусловия:** роль «Руководитель ГЦ»; КПЭ со статусом **«Ожидание утверждения плана»**.

## Переход

1. **«Необходимо утвердить плановые значения КПЭ»** или вкладка **ПФК**.

## Утверждение или отклонение

1. Фильтр **«Ожидание утверждения плана»**.
2. **«Режим редактирования»**.
3. В строке КПЭ:
   - **«Утвердить план»** (✅) → **«План утверждён»**;
   - **«Отклонить план»** (❌) → комментарий обязателен → **«План отклонён»** (есть **«Отменить отклонение»**).
4. **«Сохранить»** — зафиксировать все решения.

## Ключевые моменты

- Вы утверждаете обоснованный план для ГЦ.
- Комментарий при отклонении **обязателен**.
- Можно обработать несколько строк, затем одно **«Сохранить»**.
- После утверждения — этап ввода факта (Аналитик ФАКТ).

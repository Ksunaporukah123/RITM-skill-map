# Аналитик ФАКТ

**Предусловия:** роль «Аналитик ФАКТ»; статус факта **«Ожидание ввода факта»** (план утверждён).

## Ввод фактических значений

1. **«Необходимо внести фактические значения КПЭ»** или **ПФК**.
2. **«Полный вид»**, **«Режим редактирования»**, фильтр **«Ожидание ввода факта»**.
3. Столбец **«Факт»** → **«Сохранить»**.

**Результат:** **«Ожидание утверждения факта»** → Верификатор.

## Редактирование факта

| Исходный статус | После сохранения |
|-----------------|------------------|
| Ожидание утверждения факта | Статус без изменений |
| Факт утверждён | Подтверждение → **«Ожидание утверждения факта»** |

## Импорт из Excel

1. **Экспорт Таблица ПФК краткий вид** (с фильтрами).
2. Заполните столбец **«Факт»** только.
3. **Импорт Таблица ПФК краткий вид** → *«Успешно. Таблица обновилась!»*.

Статусы: «Ожидание ввода факта» → «Ожидание утверждения факта» при первом вводе.

## Ключевые моменты

- Сохранение только через **«Сохранить»**.
- История — **«Действие»** в строке.
- Изменение утверждённого факта — только при необходимости (повторная верификация).

# Верификатор: утверждение факта в ПФК

**Предусловия:** роль «Верификатор»; статус **«Ожидание утверждения факта»** (факт введён Аналитиком ФАКТ).

## Переход

1. **«Необходимо верифицировать фактические значения КПЭ»** или **ПФК**.

## Верификация

1. Фильтр **«Ожидание утверждения факта»**.
2. **«Режим редактирования»**.
3. В строке:
   - **«Утвердить факт»** (✅) → **«Факт утверждён»**;
   - **«Отклонить факт»** (❌) → комментарий → **«Факт отклонён»** (**«Отменить отклонение»** при ошибке).
4. **«Сохранить»**.

**После утверждения:** расчёт **значения выполнения КПЭ**; сбор данных завершён.

## Ключевые моменты

- Комментарий при отклонении обязателен.
- Несколько строк — одно **«Сохранить»**.
- Утверждённый факт — основа для оценки результативности в КР.

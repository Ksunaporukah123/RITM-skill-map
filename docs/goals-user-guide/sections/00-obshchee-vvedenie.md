# Общее введение

Документ из комплекта руководств пользователя сервиса **«Целеполагание»** в системе **РИТМ**.

## Назначение сервиса

Сервис **«Целеполагание»** предназначен для:

- постановки и согласования **карт результативности (КР)** сотрудников;
- ведения **ключевых показателей эффективности (КПЭ)** в **реестре КПЭ**;
- контроля плановых и фактических значений в **таблице ПФК**;
- **делегирования** полномочий по ролям;
- **оценки результативности** по итогам периода.

**Путь в системе:** боковое меню → **Целеполагание** (`/services/goals`).

## Структура раздела

| Вкладка | Назначение |
|--------|------------|
| **Карты результативности** | Список КР, создание, редактирование, согласование, оценка |
| **Реестр КПЭ** | Справочник показателей |
| **Таблица ПФК** | План, база/критическое отклонение, факт, статусы |
| **Делегирование** | Передача ролей сотрудникам |

**Личный кабинет** показывает **ожидаемые действия** по вашей роли. Для создания КР из контекста команды используется вкладка **«Моя команда»** в личном кабинете.

## Основные понятия

| Термин | Расшифровка |
|--------|-------------|
| **КР** | Карта результативности |
| **КПЭ** | Ключевой показатель эффективности |
| **ЗРД** | Задание по развитию |
| **ККПЭ** | Корректирующий КПЭ |
| **ПФК** | Плановые, фактические и критические значения |
| **ГЦ** | Группа целеполагания |
| **ЦОР** | Цикл оценки результативности |

**Состав КР:** КПЭ (вес 5–75%, вид, зона ответственности), ЗРД, ККПЭ при необходимости. **Сумма весов КПЭ и ЗРД = 100%.**

### Статусы карты результативности

| Статус | Смысл |
|--------|--------|
| **Черновик** | В работе у руководителя |
| **Разработка** | На акцептовании |
| **Разработка (!)** | С отклонением |
| **Активная** | Утверждена |
| **Архивная** | Завершённый период |

### Статусы плана и факта в ПФК

**План:** ожидание ввода → ожидание утверждения → план утверждён / отклонён.

**Факт:** ожидание ввода → ожидание утверждения → факт утверждён / отклонён.

## Роли и зоны ответственности

| Роль | Основные задачи |
|------|-----------------|
| **Руководитель** | КР подчинённых, акцептование, оценка |
| **Координатор** | Акцептование КР в «Разработка» |
| **Методолог ДОМРР / ДРП** | Верхнеуровневые КР |
| **Руководитель ГЦ** | Утверждение плана в ПФК |
| **Аналитик ПЛАН** | Ввод плана, импорт Excel |
| **Аналитик ФАКТ** | Ввод факта, импорт Excel |
| **Верификатор** | Утверждение факта |

Роли назначаются в **Бейз Коммандер**.

## Сквозной жизненный цикл

```
КР: Черновик → [Отправить] → Разработка → [Координатор / Методологи] → Активная
         ↑___________________________[Редактировать КР]________________________|

ПФК: Ожидание ввода плана → [Аналитик ПЛАН] → Ожидание утверждения плана
     → [Руководитель ГЦ] → План утверждён
     → Ожидание ввода факта → [Аналитик ФАКТ] → Ожидание утверждения факта
     → [Верификатор] → Факт утверждён → расчёт выполнения

КР (Активная): оценка руководителя → утверждение оценки результативности
```

## Частые ошибки

| Ситуация | Что делать |
|----------|------------|
| Нельзя выбрать КПЭ | Проверьте период карты и КПЭ |
| Не отправляется на акцептование | Сумма весов = 100% |
| «Не доступно» при переиспользовании | У сотрудника уже есть КР на период |
| Импорт Excel не проходит | Не меняйте структуру столбцов |
| Нет «Сохранить» в КР | Используйте «Редактировать КР» |
